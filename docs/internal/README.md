# Internal Development Documents (Archived)

This directory contains historical architecture notes and design documents from KAL's development process (v1–v5). They are preserved for reference but are not maintained as user-facing documentation.

For current documentation, see the [docs root](../README.md).

## Versions

- **v1/** — Initial business plan, monorepo setup, early dev guides
- **v2/** — Core + editor + engine split
- **v3/** — Architecture evolution notes
- **v4/** — Basic editor, refined engine/core docs
- **v5/** — First-principles architecture, debug system, current implementation status
